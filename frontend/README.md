# 💬 Full Stack Real-Time Chat Application

![Preview](screenshots/preview.png)


🚀 Live Demo
👉 https://fullstack-chat-app-production-345b.up.railway.app/
A modern full-stack real-time chat application built with React, Node.js, Express, MongoDB, Socket.IO, Cloudinary, and DaisyUI.

## 🚀 Features

* 🔐 Authentication & Authorization
* 💬 Real-Time Messaging with Socket.IO
* 🖼️ Image Sharing
* 🎤 Voice Notes
* 👤 User Profiles
* 🌐 Online/Offline User Status
* 🎨 32 DaisyUI Themes
* 📱 Fully Responsive Design
* ☁️ Cloudinary Media Storage
* 🍪 Secure JWT Authentication

## ⚙️ Environment Variables

Create a `.env` file in the backend directory.

```env
PORT=
MONGO_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLIENT_URL=
```

## 🖥️ Installation

### Clone Repository

```bash
git clone https://github.com/webs009/fullstack-chat-app.git
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

### Start Development Server

Backend

```bash
cd backend
npm run dev
```

Frontend

```bash
cd frontend
npm run dev
```
